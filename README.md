This is my GH pages blog. It was hacked with haste using Gatsby.

## Run

`npm run start`

## Build and deploy

I'm using the gh-pages fork and CLI tool to automatically deploy to the `gh-pages` branch which is then served. If you don't have it, you'll have to `npm install -g gh-pages`

The deploy command is already in the `package.json`. You can run it with `npm run deploy`. Or, you could, if you had my git credentials! Ha ha.
